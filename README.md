# 📊 RealSports CardCalc
**A sleek XP calculator for tracking card progression**  



## 🚀 Features
- Instant XP calculations for card upgrades  
- Clean dark-mode interface   
- Mobile-responsive design  



## 🖥️ Usage
1. Select card type  
2. Enter current XP  
3. Click "Calculate"
